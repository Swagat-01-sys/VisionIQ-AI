# 👁️ VisionIQ AI

AI-Powered Visual Intelligence Platform built using Florence-2 and Streamlit.

## Features

- Image Captioning
- Detailed Image Description
- Object Detection
- OCR (Optical Character Recognition)
- Interactive Web Interface

## Tech Stack

- Python
- Streamlit
- Florence-2
- Hugging Face Transformers
- PyTorch

## Installation

```bash
git clone <repo-url>
cd VisionIQ-AI

python -m venv venv
venv\Scripts\activate

pip install -r requirements.txt

streamlit run app.py
